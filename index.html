Styling Details

This section provides a summary of the key styling choices used throughout the design system.

Font Family

The primary font for the entire design system is **Inter**, sourced from Google Fonts. This font was chosen for its excellent readability and modern, professional look. The stylesheet imports and utilizes different font weights to create a clear visual hierarchy for headings and body text.

Key Stylings

  • **Corner Rounding:** A consistent `border-radius` of **12px** is used for main containers and cards, while buttons and form fields use **8px** for a slightly tighter, more functional look. Badges are given a full pill-shape with `9999px`.
  • **Shadows:** Subtle `box-shadow` is applied to cards and the main container (`.ds-container`) to give them a slight lift off the background, adding depth and visual separation.
  • **Transitions:** Smooth `transition` effects are used on buttons, inputs, and the overall container to create a fluid and responsive user experience when hovering or changing states.
  • **Color Variables:** All colors are defined using CSS custom properties (`--color-...`) in the `:root` pseudo-class. This allows for quick and easy color changes across the entire system by simply modifying a single variable.
